# Tailwind CSS 사용한 간단한 회사 소개 웹 페이지

이 프로젝트는 **Tailwind CSS**를 사용해 회사 소개 페이지를 구현한 프로젝트입니다. 

## 기능

- 상단에 있는 아이콘을 누르면 해당 섹션으로 스크롤됩니다
